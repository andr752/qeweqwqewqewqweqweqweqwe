## Описание программы 
Random Quote Generator (Генератор случайных цитат) - это программа, которая выводит на экран случайные цитаты, выводя списком старые цитаты, 
их можно фильтровать по теме и по автору, также очистить экран от старых цитат и сохранить их файлом.
## Требования для запуска 
Что нужно установить на компьютер: 
- Python 3.10 или выше
- Библиотеки: `tkinter`,'json','os','random'.
